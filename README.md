# ðŸ›’ Lunaria E-Commerce Platform

A modern, full-stack e-commerce web application built with **React** (frontend) and **Node.js/Express** (backend), featuring a clean UI, robust authentication, and a seamless shopping experience.

## Features

- **User Authentication:** Secure login, registration, and JWT-based session management.
- **Product Catalog:** Browse, search, and filter products by category (Men, Women, Kids, etc.).
- **Shopping Cart:** Add, update, and remove items; persistent cart state.
- **Checkout & Orders:** Address management, payment card storage, and order placement.
- **Admin Dashboard:** Product management, CSV bulk upload, and order overview.
- **Responsive Design:** Mobile-friendly, fast, and accessible UI.
- **Chatbot Assistant:** Built-in virtual assistant for FAQs and support.
- **Profile Management:** Update user info, manage addresses and payment methods.
- **Order History:** View past orders and order details.
- **Secure Backend:** Express.js REST API, MongoDB models, and authentication middleware.

## Tech Stack

- **Frontend:** React, React Router, Context API, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT
- **Other:** Axios, Multer (file uploads), Framer Motion (animations)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bunny0417/InternshipProject.git
   cd lunaria-ecommerce
   ```

2. **Install dependencies:**
   - For the backend:
     ```bash
     cd server
     npm install
     ```
   - For the frontend:
     ```bash
     cd ../client
     npm install
     ```

3. **Set up environment variables:**
   - Copy `.env.example` to `.env` in the `server/` directory and fill in your MongoDB URI and JWT secret.

4. **Run the app:**
   - Start the backend:
     ```bash
     cd server
     npm start
     ```
   - Start the frontend:
     ```bash
     cd ../client
     npm run dev
     ```

5. **Visit:**  
   - Frontend: [http://localhost:5173](http://localhost:5173)  
   - Backend API: [http://localhost:5000](http://localhost:5000)
